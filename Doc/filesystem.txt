DOGEFS:
The first selector is contained no message.
The second selector is a infomation area:

QWORD [0x000]: Identifier #1
    Must be all 0x5a

WORD  [0x008]: SIPL Size (<=63)

WORD  [0x00a]: SIPL Location (Logical Sector)

DWORD [0x00c]: FS size (sectors) low 32 bits

WORD  [0x010]: FS size (sectors) high 16 bits

WORD  [0x012]: Operating system size (sectors)

DWORD [0x014]: Operating system location (Logical sector)
